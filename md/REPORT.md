# HyperDesk Dashboard & Command Center - Development Summary

HyperDesk를 단순한 VM 관리기에서 **Enterprise-Grade Command Center**로 변모시키기 위한 주요 업그레이드 내역입니다.

## 1. 정밀 텔레메트리 (High-Precision Telemetry)
*   **스파크라인 고도화**: 레거시 SVG 대신 `recharts` 라이브러리를 도입하여 부드러운 Cubic-Spline 곡선의 실시간 차트를 구현했습니다.
*   **시스템 지표 세분화**: CPU 부하량, 메모리 할당량, 네트워크 I/O(KB/s)를 실시간으로 추적 및 시각화합니다.
*   **실시간 지연시간(Latency) 추적**: 원격 자산(MST/VDI)에 대해 단순 On/Offline이 아닌 **실시간 응답 시간(ms)**을 측정하여 네트워크 성능을 모니터링합니다.

## 2. 인터랙티브 레이아웃 & 애니메이션
*   **물리 기반 드래그 앤 드롭**: `framer-motion`의 `Reorder` 컴포넌트를 사용하여 VM과 원격 자산 리스트의 순서를 자유롭게 재정렬할 수 있습니다.
*   **상태 유지 동기화**: 백엔드에서 텔레메트리 데이터를 수신할 때 사용자가 조작 중인 리스트의 순서가 튀지 않도록 로컬 상태(Local State)와 서버 데이터를 매칭하는 싱크 로직을 완성했습니다.
*   **자동 감지(AUTO) 로직**: 윈도우 레지스트리에 기록된 RDP 기록 및 Omnissa/VMware Horizon 서버 설정을 자동 스캔하여 리스트에 즉시 반영하는 Smart Discovery 기능을 탑재했습니다.

## 3. 산업용 랙(Rack) 테마 디자인
*   **Modular Rack 디자인**: 모든 자산 라인에 'Industrial Grip/Ear' 효과를 추가하여 마치 실제 서버 랙을 조작하는 듯한 하드웨어적인 감성을 부여했습니다.
*   **가독성 중심 UI**: 복잡한 정보 밀도를 유지하면서도, 오프라인 자산은 Dim 처리하고 중요 지표는 고대비 폰트와 글로우 효과를 사용하여 스캔 효율을 극대화했습니다.
*   **프로토콜 전용 배지**: RDP, HORIZON, VDI 등 각 자산의 연결 타입을 직관적으로 구분하기 위한 전용 테크니컬 배지를 추가했습니다.

## 4. 백엔드 아키텍처 (Tauri/Rust)
*   **비계층적 명령 인터페이스**: PowerShell 자동화 모듈을 강화하여 Hyper-V 관리 모듈 없이도 기본적인 상태를 읽을 수 있도록 폴백(Fallback) 처리했습니다.
*   **네트워크 프로빙 엔진**: TCP Connect 타임아웃을 통한 지연시간 측정 엔진을 구축하여 미세한 품질 저하를 사전에 인지할 수 있는 기반을 마련했습니다.

***

> [!TIP]
> 이제 리스트의 순서를 드래그하여 최우선 순위 장비를 최상단에 배치하고, 지연시간이 높아지는 장비를 실시간으로 감시할 수 있습니다.
